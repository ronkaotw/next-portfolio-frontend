# Next Portfolio Frontend

## Description
Personal portfolio website built with Next.js and Tailwind CSS. Integrates with `portfolio-api` to display projects, contact info, and blog posts.

## Features
- Fully responsive UI
- Dynamic fetching from backend API
- Smooth animations and transitions

## Tech Stack
- React
- Next.js
- Tailwind CSS
- Framer Motion
- Axios / Fetch API for data

## Installation
```bash
git clone https://github.com/ronkaotw/next-portfolio-frontend.git
cd next-portfolio-frontend
npm install
npm run dev
